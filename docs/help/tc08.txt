TC08 DECtape quick reference
============================

Attach and status:
  dt0|dt1 att|attach <file> [new]
  show dt

Block I/O:
  dt0|dt1 read <block> <addr>
  dt0|dt1 write <block> <addr>
    - blocks are 129 words (128 data + checksum forced to 0)
    - args are octal

IOT devices:
  Control: device 076, Data: device 077

IOT codes:
  IOT    Mnemonic  Description
  06762  DTCA      Clear transfer address
  06764  DTSF      Skip if controller ready
  06766  DTLB      Read block into memory at transfer address
  06771  DTXA      Load transfer address from AC
